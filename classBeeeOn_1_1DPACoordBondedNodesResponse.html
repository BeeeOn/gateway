set< uint8_t > DPACoordBondedNodesResponse::decodeNodeBonded ( ) const

Each node id of device is stored as bit index.

Example:

  • PData: FE.01.00.00... (32B)

List of node id:

  • 1 = 0 * 8 + 1
  • 2 = 0 * 8 + 2
  • 3 = 0 * 8 + 3
  • ...
  • 8 = 1 * 8 + 0
