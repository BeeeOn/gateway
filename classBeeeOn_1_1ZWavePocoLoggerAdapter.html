Detailed Description

ZWavePocoLoggerAdapter adapts the the logging infrastructure of the OpenZWave library to the Poco::Logger. It converts the OpenZWave logging levels as follows:

  • Fatal - PRIO_FATAL
  • Error - PRIO_ERROR
  • Warning - PRIO_WARNING
  • Alert - PRIO_NOTICE
  • Always - PRIO_INFORMATION
  • Info - PRIO_INFORMATION
  • Detail - PRIO_DEBUG
  • Debug - PRIO_DEBUG
  • StreamDetail - PRIO_TRACE
  • Internal - PRIO_TRACE

Each created Poco::Message would also contain named parameter "node" holding the node ID. This can be used for better formatting of log messages.
