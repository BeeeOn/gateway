Detailed Description

Explore configuration files guided by the given configuration. It looks for keys:

  • application.configDir - in this directory, it processes files in config.d/
  • application.extended.configDir - processes all files in this directory
  • application.extended.configFiles - processes files specifies in this list separated by a platform specific separator (e.g. ':')
