Detailed Description

Serves to read and verify data written by the DataWriter.

Invalid data (wrong format or wrong checksum) from the input stream are skipped.

string DataReader::next ( )
overridevirtual

Serves to access next data from the input stream.

However you can access the data through this method without calling the method hasNext(), it is recommended to ensure if there are next valid data, because if there are not, this method throws an exception.

Returns
If there are already loaded data, returns them, else tries to load next data and in the case of success, returns them.
Exceptions
IllegalStateExceptionin the case of no next valid data are available.
