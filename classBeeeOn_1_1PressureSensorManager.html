Detailed Description

Manager for BeeeOn gateway's internal air pressure sensor. Actuall value is obtained by reading from specified file typically located in sysfs.

This class instance is restricted for managing only one physical device with one module representing pressure.

Measured values are reported periodically with specified refresh time.

This file is typically named "pressure0_input"
